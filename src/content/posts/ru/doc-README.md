Xingluo — это современная CMS для блогов, созданная на [Astro](https://astro.build/) с визуальным стилем [shadcn/ui](https://ui.shadcn.com/). Она обеспечивает более современный визуальный опыт благодаря плоским элегантным компонентам shadcn и цветовой системе OKLCH, а также включает систему комментариев, опциональную поддержку MDX и аудио/видеоплееры.

## Основные возможности

- **Производительность высшего уровня**: статическая генерация Astro, встроенные SVG-иконки во время сборки (ноль JS во время выполнения), ленивая загрузка комментариев и плееров, очистка orphan-ассетов
- **Современный внешний вид**: компоненты shadcn/ui new-york, цветовое пространство OKLCH, плавный темный режим (защита FOUC)
- **Многоязычность**: перевод на уровне UI и контента, маршрутизация `prefixDefaultLocale: false`, SEO-объявления hreflang и x-default
- **Улучшение контента**: опциональный MDX, подсветка кода Shiki с двойной темой, callouts, сворачиваемое оглавление, прокручиваемые таблицы
- **Время чтения**: интеллектуальная оценка (CJK по количеству символов, латиница по количеству слов), отображается на карточках и страницах деталей
- **Связанные посты**: автоматически рекомендуются по общим тегам
- **Категории постов**: назначаются через frontmatter, с отдельными страницами категорий и навигацией
- **Фиксированная боковая панель TOC**: фиксированное оглавление справа на больших экранах, отслеживание прокрутки IntersectionObserver
- **Система комментариев**: giscus / twikoo / waline, с учетом темы, ленивая загрузка
- **Медиаплееры**: аудио APlayer и видео DPlayer, с точками входа MD fence и компонента MDX
- **Поиск**: полнотекстовый поиск Flexsearch, индексы по языкам, сохранение состояния View Transitions
- **Полное SEO**: динамические OG-изображения (satori + sharp), RSS, карта сайта, структурированные данные JSON-LD (BlogPosting + BreadcrumbList), нормализация canonical
